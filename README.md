# A terminal based music player #
Made with the curses module in Python.
## Requirements ##
* Python 3.8
* curses
* python-vlc
## Keys ##

* p - pause or play 
* ctrl + backspace - navigate backwards
* enter - navigate forwards (enter directory) or start playing all music in folder from the selected song
